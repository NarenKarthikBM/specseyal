# 000-sample — Overview

> Fixture. Non-technical deck. One page. This is what a human reads at the council gate,
> and what the GUI renders on a phone (D21).

**What we're building.** A worked example of every document the pipeline produces — a reference
copy, so we can tell later whether the pipeline's output is shaped correctly.

**Why.** We just wrote down the rules for what each document must contain. Nobody has yet written a
document that follows them. Until one exists, the rules are untested.

**What could go wrong.** The example and the rules drift apart, and the example stops being a
reference. We prevent this by requiring that any change to the rules updates the example in the
same change.

**What it costs.** Nothing. No AI sessions run, no code is written.

**What "done" looks like.** Every document listed in the rules exists here, and each one follows
its own rule.
